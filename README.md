# reward-punishment-module
臺中市立大甲國民中學 - 作業抽查管理系統 (獎懲建議表模組)。支援自動化分頁列印、獨立建檔與批次 PDF 下載，專為提升行政效率打造之後端分離系統。
💡 歷史版本沿革 (Changelog)
v8.1.0：旗艦優化版，全面正名，新增 API 網址防呆機制與 UX 微動畫。

v8.0.0：架構大升級，全面轉換為前後端分離 (GitHub Pages + GAS API) 模式。

v6.4.0：回歸一班一檔獨立建檔，強化批次 PDF 下載穩定度。

v6.0.0：導入 Auto-Pagination (自動分頁) 引擎，徹底解決列印跨頁跑版問題。

v5.9.0：導入 RichText 雙重字體技術，精細化表頭公文排版。
